CHALLENGES

When doing these, you might like to play with them in codepen:

  • - Sign up to a password manager.
  • - Investigate what a css reset is and implement it on your site.
  • - CSS hover effects are a great way to give feedback. Add one to your nav items.
  • - Make your hover effect a bit nicer by adding a css transition of around 0.5 second.

EXTRA CHALLENGES

In case you want more!

  • - Use a google font (pick a font, follow the instructions, cut/paste)
  • - Make your website responsive so that it reads well on mobile, tablet and desktop (google 'css breakpoints')
  • - Make your nav items more "button-like" by putting them in a div with a height and width. Make the whole div clickable

BONUS SUPER EPIC HARD CHALLENGE

There is a way to continuously deploy changes to your site as you update. This is done by signing up to Github and getting Netlify to deploy from your repository there. Git can be a bit tricky, so I recommend downloading the Github desktop app.

That's how I've done this site. You can check it out here!
