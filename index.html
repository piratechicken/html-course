Design resources

  • A warning for all budding web devs: The Worst Website Ever (there's a little link in there somewhere that lists all the errors)
  • Dribble is a great place to get design inspration
  • Pinterest can also be great. WHen making a design, you can create a board to keep ideas and inspirations in one place. You might also create a board for websites that have great design.
  • A lot of web designers create a mood board as part of the design process. You can display your colors, fonts, logos.
  • Don't forget to create mockups in Figma. You can play around to find something that works before you start to code. Just don't forget to keep in mind the constraints of HTML/CSS styling.

CHALLENGES

(when doing these, you might like to play with them in codepen): - Sign up to a password manager! - Investigate (ie google) what a css reset is and implement it on your site - CSS hover effects are a great way to give feedback. Add one to your nav items - Make your hover effect a bit nicer by adding a css transition of around 0.3 second

EXTRA CHALLENGES

in case you want more! in case you want more!

- Use a google font (pick a font, follow the instructions, cut/paste): https://fonts.google.com/ - Make your website responsive so that it reads well on mobile, tablet and desktop (google css breakpoints) - Make your nav items more "button-like" (don't use html button) and make the whole button clickable (with the hover effect and transition)
