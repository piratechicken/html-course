CHALLENGES

(when doing these, you might like to play with them in codepen): - Sign up to a password manager! - Investigate (ie google) what a css reset is and implement it on your site - CSS hover effects are a great way to give feedback. Add one to your nav items - Make your hover effect a bit nicer by adding a css transition of around 0.3 second

EXTRA CHALLENGES

- Use a google font (pick a font, follow the instructions, cut/paste): https://fonts.google.com/ - Make your website responsive so that it reads well on mobile, tablet and desktop (google css breakpoints) - Make your nav items more "button-like" (don't use html button) and make the whole button clickable (with the hover effect and transition)
